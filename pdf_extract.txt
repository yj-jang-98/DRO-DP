II. REVIEW OF DIFFERENTIAL PRIVACY
We introduce the basic notions of(ϵ, δ)-differential privacy
(DP) [1]. The core idea is to design a randomized mecha-
nism whose output distribution is nearly indistinguishable for
adjacent input datasets, thereby preventing an adversary from
inferring the input based on the observed output.
To formalize this, letDdenote the set of all input datasets
under consideration. We fix a distance metricDist:D×D →
Rand define the set of adjacent dataset pairs as
Adj:={(d, d′)∈ D × D |Dist(d, d′)≤γ},(1)
whereγ >0is a tunable parameter. The quantity to be
computed for each dataset and released publicly is modeled
by a deterministic queryq:D → Q, whereQ ⊂Rhfor
simplicity. To preserve privacy, we do not release the query
directly. Instead, we employ a randomized mechanismMthat,
for eachd∈ D, outputs aQ-valued random variableM(d).
Based on these,(ϵ, δ)-DP is defined as follows.
Definition 1.For givenϵ≥0andδ∈(0,1), a randomized
mechanismMsatisfies(ϵ, δ)-DP if
P[M(d)∈S]≤exp(ϵ)·P[M(d′)∈S] +δ
for allS⊂ Qand(d, d′)∈Adj. Ifδ= 0, the mechanism is
said to satisfyϵ-DP.□
The privacy budgetϵgoverns the maximum allowable
change in the output probability distribution for adjacent input
datasets, where a smallerϵcorresponds to a stronger privacy
guarantee. The parameterδrelaxes this by allowing a failure
probability of at mostδ. Thus,ϵ-DP is a stronger condition
and implies(ϵ, δ)-DP for anyδ≥0.
An appealing feature of DP is its immunity to post-
processing. That is, applying any transformation on the mecha-
nism’s output, such as for feedback control or state estimation,
does not degrade the established privacy level. Additionally,
DP admits an additive composition rule that enables the
characterization of cumulative privacy loss when multiple
queries are released for a dataset. This feature is particularly
advantageous when applying DP to dynamical systems, as
it facilitates a rigorous accounting of the total privacy loss
incurred over time. For a comprehensive treatment of these
properties and other aspects of DP, we refer the reader to [1].

Laplace mechanisms that are commonly used in the DP
literature. The Gaussian mechanism ensures privacy by addingnoise drawn from the multivariate Gaussian distribution to

III. PROBLEMFORMULATION
Consider a discrete-time plant written by
x(k+ 1) =Ax(k) +Bu(k) +w(k),(4)
y(k) =Cx(k),
wherex(k)∈Rnis the state with the initial valuex(0)∼

is the output, andu(k)∈Rmis the input. The process
noisew(k)∈Rnis assumed to be white Gaussian with
w(k)∼ N(0,Σ w), whereΣ w≻0. Throughout, the pa-
rameters{A, B, C,Σ w,Σini, xini}are considered as public
information.
In this work, we treat the state trajectoryx 0:N over a
finite horizonN∈Nas sensitive information that the plant
wishes to protect. The adversary is modeled as either an

eavesdropper or the controller itself (which is to be designed)
that observes the output trajectory and attempts to estimate the
state trajectory. To counteract this, the plant does not release
the raw outputy(k). Instead, it publishes a privatized output
˜y(k) =y(k) +v(k),
wherev(k)∈Rpis an artificial measurement noise.
The objective of the plant is to design the measurement
noise sequencev 0:N using either the Gaussian or Laplace
mechanism such that the randomized mapping from the state
trajectory to the privatized output trajectory˜y 0:Nis(ϵ, δ)-DP.
While the desired privacy level(ϵ, δ)is assumed to be publicly
known, we crucially assume that the specific mechanism or
noise parameters chosen by the plant remains unknown to the
adversary.
Simultaneously, the controller aims to synthesize an optimal
output feedback policy relying solely on available public
information. A fundamental challenge is that the distribution
ofv(k)is not available because the underlying mechanism
is unknown, which makes standard optimal control methods
such as LQG inapplicable. To address this, we formulate a
minimax optimal control problem that seeks to minimize the
worst case expected cost functional over the set of admissible
noise distributions that ensure(ϵ, δ)-DP, thereby ensuring
robust performance. Although we focus on Gaussian and
Laplace mechanisms for simplicity, the proposed framework
can be naturally extended to incorporate various other noise
mechanisms.
Specifically, the proposed framework is established through
the following sequential steps:
1) We first formalize the notion of DP for dynamical systems
and derive conditions on the noise parameters under
which the Gaussian and Laplace mechanisms satisfy
(ϵ, δ)-DP.
2) We then construct a KL-divergence ball that contains the
PDFs of all admissible noise distributions and formulate
a tractable minimax optimal control problem.
3) Finally, we synthesize an optimal controller by leveraging
techniques from risk-sensitive control to solve the mini-
max problem, and analyze the trade-off between privacy
guarantee and control performance.
