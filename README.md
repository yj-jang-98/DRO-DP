# Distributionally Robust Optimal Control for Differentially Private Dynamical Systems

This repository contains the simulation code for the paper:
**"A Distributionally Robust Optimal Control Approach for Differentially Private Dynamical Systems"** by Yeongjun Jang, Kaoru Teranishi, and Junsoo Kim.

## ⚙️ Simulation Details

1. Compares the proposed method against a standard LQG controller
2. Analyze the performance of the proposed method under varying privacy parameters

## 🚀 Usage

Run main.ipynb
